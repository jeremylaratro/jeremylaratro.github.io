This website

This website uses almost exclusively HTML and CSS, at this point. The website is hosted by GitHub Pages and the domain is routed via AWS (route55). I do plan on implementing javascript in the future, but wanted to take a quick dive into web design, while also being able to quickly create a site where I can host my portfolio and my personal projects, without overwhelming myself while learning Python. The first implementation of this site was using a web design platform, but I felt that it was not fulfilling in that I was not truly learning the ins and outs of web design. I decided to completely scrap the website and essentially start from scratch. Keeping the site as just CSS and HTML allowed me to get the website up and running in about 3 (long) days, while also learning a massive amount of information. There is minimal javascript in use, mainly for opening these collapsible boxes, but the majority of the code consists of css elements and html. I mainly used a grid system to create the basic layout, and then used containers, like this one, to hold all the content. You can browse through the source code for the entire website here
